Trenti Data Governance

Define a Data Governance program for your organization.

Available languages

The user interface only has a Spanish version in its current state. It does not support English.

Defined data governance framework

The module relies on the solid Trenti Data Governance framework, defined by Ph.D Juan Rafael Yebenes Serrano in his thesis, and has been adapted for meeting the needs of SMEs.

This module contains the following components: roles, data governance objectives, actors, KPIs, principles, data quality property measures, policies, standards, procedures, input parameters, information assets, data sources, data entities, data elements, glossary terms, data quality rules, data quality characteristics, data quality requirements, security classifications, categories, locations and organization units.

The aforementioned thesis can be found in: http://hdl.handle.net/10902/25087.

Features

  • Manage your Data Governance program.
  • Query the existing instances with custom filters, groupings and text search.
  • Get all the details of an instance or add a new one with form views.
  • Export all the information organized in tables to Excel with a single click.

Development team

The Trenti Data Governance module has been developed within the University of Cantabria as part of a Final Degree Project.

Specifically, it has been built in the Real-Time and Software Engineering research group.

Authors: Ivan Ortiz del Noval, Juan Rafael Yebenes Serrano and Marta Elena Zorrilla Pantaleon.

Rights transferred to the University of Cantabria, 2023.
